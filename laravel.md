# php-laravel-commands

List commands
```
php artisan

```

Create project
```
composer create-project laravel/laravel <projectname>

```

Run Server
```
php artisan serve --host=<host> --port=<port>

```

Create controller
```
php artisan make:controller <controllername> --resource

```

Run Server
```
php artisan make:component <componentname>

```

Run Server
```
php artisan make:component <componentname>

```
